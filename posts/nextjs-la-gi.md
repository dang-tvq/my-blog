# Next.js là gì? 🚀

**Next.js** là một framework xây dựng trên React, được phát triển bởi Vercel. Nó giúp bạn tạo ra các ứng dụng web hiện đại một cách dễ dàng hơn.

## Những tính năng nổi bật

### 1. Server-Side Rendering (SSR)
Trang web được render trên server, giúp tải nhanh hơn và SEO tốt hơn.

### 2. Static Site Generation (SSG)
Build toàn bộ trang thành HTML tĩnh — siêu nhanh!

### 3. App Router
Hệ thống routing mới dựa trên folder structure, rất trực quan.

## Ví dụ code đơn giản

```tsx
export default function HelloPage() {
  return <h1>Xin chào từ Next.js!</h1>
}
```

## Kết luận

Next.js là lựa chọn tuyệt vời cho bất kỳ dự án web nào, từ blog cá nhân đến ứng dụng doanh nghiệp lớn.
